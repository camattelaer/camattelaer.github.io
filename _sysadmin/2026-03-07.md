I built this setup by collecting old hardware over the years, oaccompanied by purchases of both new and used parts. One of the used parts is the HP ML350 gen 10 system (middle). Unfortunately, I kept encountering power brake issues when installing GPUs in this machine (something I haven't been able to fix to date). This issue forced me to setup a Windows Subsystem for Linux (WSL) VM to provide GPU compute capabilities.

In addition, I had other GPUs available which did not fit in my personal Windows system (since it also houses a RAID card). I bought a used Intel i7 6800K and an X99 motherboard and combined those into a dedicated GPU node. The result, after transferring the hardware into rackmounted cases, is version 1.0 of my own little home compute node!

I am currently working on v2.0. When this update is finished, I will also gives a bit more details about each individual node.

Stay tuned!
